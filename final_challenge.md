# 最終課題：学生成績管理システム

## 課題の説明
学生の成績を管理する簡単なプログラムを作成してください。

## 機能要件

### 1. 学生情報の管理
- 学生の名前、年齢、点数を管理する
- 最大5人の学生を登録できる

### 2. 実装する機能
1. **学生の登録**
   - ユーザーから学生情報（名前、年齢、点数）を入力
   - 配列に保存

2. **全学生の表示**
   - 登録されているすべての学生の情報を表示
   - 番号付きで表示

3. **平均点の計算**
   - 全学生の平均点を計算して表示

4. **最高点の学生を表示**
   - 最も点数が高い学生の情報を表示

5. **メニュー表示**
   - ユーザーが機能を選択できるメニューを表示
   - 終了するまで繰り返す

## 使用する技術
- 構造体（学生情報を管理）
- 配列（複数の学生を管理）
- ループ（メニューの繰り返し、配列の処理）
- 関数（各機能を関数に分ける）
- ポインタ（関数で配列を操作）

## ヒント

### 構造体の定義
```c
struct Student {
    char name[50];
    int age;
    int score;
};
```

### 必要な関数（例）
- `void addStudent()` - 学生を登録
- `void showAllStudents()` - 全学生を表示
- `void showAverage()` - 平均点を表示
- `void showTopStudent()` - 最高点の学生を表示
- `void showMenu()` - メニューを表示

### グローバル変数（使ってもOK）
```c
struct Student students[5];  // 学生の配列
int studentCount = 0;        // 現在の学生数
```

### scanf の使い方
```c
int choice;
scanf("%d", &choice);  // 整数を入力

char name[50];
scanf("%s", name);  // 文字列を入力（スペースなし）
```
