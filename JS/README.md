# JavaScript学習教材

C言語経験者向けのJavaScript学習教材です。HTML + JavaScriptで、ブラウザで直接実行できます。

## 📚 学習カリキュラム

### 基礎編
1. **01-variables-and-types.html** - 変数と型
   - C言語との違い（動的型付け）
   - let, const, var
   - データ型と型変換
   - テンプレートリテラル

2. **02-operators-and-control.html** - 演算子と制御構文
   - 算術演算子（除算の違いに注意）
   - 比較演算子（== vs ===）
   - if, switch, for, while
   - FizzBuzz、素数判定などの課題

3. **03-functions.html** - 関数
   - 第一級オブジェクトとしての関数
   - 関数宣言、関数式、アロー関数
   - デフォルト引数、可変長引数
   - 高階関数

4. **04-arrays-and-objects.html** - 配列とオブジェクト
   - 配列の操作（push, pop, shift, unshift）
   - 高階メソッド（map, filter, reduce）
   - オブジェクトの作成と操作
   - 配列とオブジェクトの組み合わせ

### フロントエンド編
5. **05-dom-basics.html** - DOM操作の基礎
   - 要素の取得（querySelector）
   - 内容の変更（textContent, innerHTML）
   - スタイルの変更
   - 要素の作成と追加

6. **06-events.html** - イベント処理
   - イベントリスナーの登録
   - マウスイベント、キーボードイベント
   - イベントオブジェクト
   - フォームイベント

### 実践編
7. **07-final-project.html** - 総合演習
   - ToDoリスト
   - 簡易計算機
   - 学生成績管理システム
   - タイマーアプリ（チャレンジ）

## 🚀 使い方

1. 任意のHTMLファイルをブラウザで開く
2. 解説を読む
3. 課題のコードを書く
4. ボタンをクリックして実行
5. ブラウザの開発者ツール（F12）でコンソールを確認

## 💡 学習のポイント

### C言語との主な違い
- **型システム**: 動的型付け（変数宣言時に型を指定しない）
- **メモリ管理**: ガベージコレクション（自動）
- **関数**: 第一級オブジェクト（変数に代入可能）
- **配列**: 可変サイズ、異なる型も格納可能
- **除算**: 常に小数を返す（整数除算ではない）
- **比較**: === を使う（型変換なしの厳密比較）

### 推奨される学習順序
1. 01から順番に進める
2. 各ファイルの課題を必ず自分で書く（穴埋めではなく1から）
3. エラーが出たら、ブラウザのコンソールを確認
4. 理解できたら次のファイルへ
5. 07の総合演習で全体を復習
